# Golang Backend API – Learning Project

This is a beginner-friendly Golang backend API project.
I am building this project step by step to learn real-world backend development.

## 🚀 Features
- REST APIs using net/http
- JSON request & response
- Input validation
- Common API response structure
- Signup API with password hashing (bcrypt)
- Clean folder structure
- Git version control
